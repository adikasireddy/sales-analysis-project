# 📊 Sales Data Analysis Project

## 📌 Overview
This project analyzes sales data using Python to generate insights and visualizations.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📊 Key Features
- Daily revenue trend analysis
- Product-wise revenue comparison
- Region-wise sales distribution
- Top-performing products identification

## 📈 Output Visualizations
- Line Chart (Revenue Trend)
- Bar Chart (Product Sales)
- Pie Chart (Region Distribution)

## 💡 Business Insights
- Identified best-selling product
- Found top-performing region
- Calculated total and average revenue

## ▶️ How to Run
```bash
python analysis.py
